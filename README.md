# Databricks Data Engineering Project

## Overview
End-to-end data engineering pipeline built using Databricks Community Edition.

## Architecture
Raw → Bronze → Silver → Gold

## Tech Stack
- Databricks Community Edition
- PySpark
- Delta Lake
- SQL

## Pipeline Flow
1. Raw data ingestion
2. Data cleansing and transformation
3. Aggregations and analytics
4. Data quality validation

## Execution
Run notebooks sequentially from ingestion to analytics.
